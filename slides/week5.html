Transitions

Simple animation from point A to B

Properties


transition: width 1s ease 1s;

/* is shorthand for */
transition-property: width;
transition-duration: 1s;
transition-timing-function: ease;
transition-delay: 1s;

Triggers

Any change to an element's properties.

Pseudo-classes such as :hover, :focus, and :valid.

Styles applied through media queries.

transition-property

Animatable properties only.

Generally only between absolute values.

Once again, these can be in a comma separated list.



#div1 {
	transition-property: width, height, transform;	
}

transition-duration

Duration of the transition.

Defaults to 0 - always specify this.

transition-timing-function

Give life to your transitions.

Create complex eases with cubic-bezier.com/

transition-delay

Can be used for simple sequencing.

Backgrounds

Decoration Only

Backgrounds are not accessible. Put images relevant to the content in an img tag.

The Properties


#div1 {
	background-color: transparent;
	background-image: url('path/to/image.png');
	background-repeat: repeat;
	background-position: 0% 0%;
	background-attachment: scroll;
	background-size: auto;
}

background-image

Used for large images, patterns, and gradients

background-repeat

repeat, no-repeat, repeat-x, repeat-y

Neat effects!

background-position


#div1 {
	background-image: url('path/to/image.png');
	background-position: center top;
}
#div2 {
	background-image: url('path/to/image.png');
	background-position: 10% 20%;
}
#div3 {
	background-image: url('path/to/image.png');
	background-position: 30px 80px;
}

defaults to top left

background-size

length, percentage, cover, contain

Undeclared values are set to auto

Be careful to maintain your proportions


#div1 {
	background-size: 200px 400px;
}
#div2 {
	background-size: 100%;
}
#div3 {
	background-size: cover;
}

background-attachment

scroll, fixed


#div1 {
	background-attachment: scroll;
	/* Default, scrolls with the page */
}
#div2 {
	background-attachment: fixed;
	/* Relative to viewport, like fixed positioning */
	/* Does not work on mobile!!! */
}

Layering

Can comma separate multiple backgrounds

Use for textures or effect layering



#div1 {
	background-image: 
		url('top-image.png'),
		url('next-image.png');
	background-repeat: repeat, no-repeat;
	background-size: auto, cover;
}
Patternify for Backgrounds

Fast Loading

Load smaller images first and blur them.


.big-image {
	background-image: url('smaller-image.png');
	filter: blur(10px);

	&.loaded {
		background-image: url('bigger-image.png');
		filter: blur(0);
	}
}

background-blend-mode

Blends layers of background images and color.

Like Photoshop blend modes.


.layered-image {
	background-color: red;
	background-image: url(some-img.jpg);
	background-blend-mode: multiply;
}
